> **LUMIÈRE LUXE** is an ultra-luxurious, state-of-the-art e-commerce experience crafted for connoisseurs of high-end horlogerie, haute couture, fine jewelry, and rare leather goods. Showcasing world-renowned maisons including **Rolex, Patek Philippe, Audemars Piguet, Cartier, Louis Vuitton, Gucci, Hermès, and Balenciaga**.

---

## ✨ Key Features & Architectural Highlights

- 💎 **Obsidian Gold Design System:** Crafted with a dark obsidian palette (`#08080a`), satin champagne gold accents (`#D4AF37`), metallic shimmer typography, and dynamic glassmorphic cards.
- ⌚ **Haute Horlogerie 360° AR Simulator:** Interactive product detail modal revealing full watch mechanical movements, caliber specs, gemstone clarity, and multi-angle galleries.
- ⚖️ **Piece Comparison Matrix:** Compare horlogerie specifications of two rare luxury pieces side-by-side in real time.
- 🛍️ **Haute Shopping Bag & VIP Checkout:** Dynamic subtotal calculations, promo code system (`VIPLUXE10` for 10% VIP discount), Centurion Black Card / Swiss Wire / Crypto payment simulation with celebratory confetti.
- 🤵 **VIP Stylist Concierge Drawer:** Interactive assistant drawer for personalized luxury acquisition consultation.
- 🔍 **Real-Time Filtering Engine:** Filter instantly by category, prestgious brand, search keywords, and price threshold slider up to $350,000+ USD.

## 🚀 Quick Start & Local Setup

### 1. Clone the repository
```bash
git clone https://github.com/nityapatel18/lumiere-luxe-premium.git
cd lumiere-luxe-premium
```

### 2. Install dependencies
```bash
npm install
```

### 3. Launch Development Server
```bash
npm run dev
```
Open `http://localhost:5173` in your browser to experience the platform.

### 4. Build for Production
```bash
npm run build
```
